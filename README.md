# 🌱 EcoWorld — Made by Naitik

Welcome to my very first interactive webpage! 🚀 

**EcoWorld** is a modern, responsive landing page designed to celebrate nature and raise awareness about protecting our environment. The page features a premium dark theme, glassmorphism elements, and ambient glowing backdrops using my five favorite colors.

---

## 🎨 Color Palette & Themes

This webpage is built around my favorite colors, each representing a key element of our environment:

* 🟢 **Forest Green** (`#10b981`) — Representing our forests, vegetation, and land ecosystems.
* 🔵 **Ocean Blue** (`#06b6d4`) — Representing water, marine life, and global moisture regulation.
* 🌸 **Blossom Pink** (`#ec4899`) — Representing flower biodiversity and pollinators.
* 🟡 **Sun Yellow** (`#f59e0b`) — Representing light, warmth, and solar energy.
* 🟠 **Sunset Orange** (`#f97316`) — Representing sunset rays and clean, renewable heat.

---

## ✨ Features

### 1. 🌸 Interactive Virtual Garden
Select your favorite color from the control bar and click anywhere inside the garden canvas area. Watch dynamic, animated flowers sprout and bloom right where you clicked!
* *Note: The garden holds up to 40 flowers. Spawning more will clean up the oldest flowers automatically to keep your screen fast and clean.*

### 2. 💡 Eco-Tip Generator
Click the **"Get Another Tip"** button to cycle through ten environmental tips. Features smooth text fade-in and slide-up micro-animations.

### 3. 🖼️ Glassmorphism Grid Layout
Four custom card grids outlining critical ecosystems on Earth, complete with responsive layouts, icon displays, and interactive lift transitions on hover.

## 🚀 How to Open and Run

Since this is a lightweight, frontend webpage (written in pure HTML, CSS, and JavaScript), you don't need any server configuration or installers!

1. Download or clone this repository to your computer.
2. Locate the file **`index.html`** in your project folder.
3. Double-click **`index.html`** (or right-click and choose *Open with*) to run it in your favorite web browser (Chrome, Edge, Safari, Firefox).
